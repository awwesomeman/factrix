# Get Started

factrix answers one question: **「這個因子在統計上真的有效嗎？」**

It produces a single `primary_p` per factor cell from a NW HAC-corrected canonical procedure (IC / FM-λ / CAAR / TS-β). Feed surviving factors into downstream portfolio constructors — factrix does not backtest or construct portfolios.

- [Installation](install.md) — install via `uv` or `conda`
- [Quickstart](quickstart.md) — 30-second smoke test and factory lookup
- [Concepts](concepts.md) — three orthogonal axes and five analysis scenarios
